# Flutter APP 开发环境安装指南

## 概述

本指南将帮助您安装运行缅甸房产平台 APP 所需的所有软件。

**预计时间**: 30-60分钟
**所需空间**: 约 10GB 硬盘空间

---

## 第一步：安装 Java JDK

### 为什么需要？
Android 开发需要 Java 环境。

### 安装步骤

1. **下载**
   访问: https://learn.microsoft.com/java/openjdk/download
   下载: **Microsoft Build of OpenJDK 17** (Windows x64)

2. **安装**
   双击下载的 `.msi` 文件，按向导完成安装
   建议安装路径: `C:\Program Files\Microsoft\jdk-17`

3. **验证**
   打开新的命令提示符，输入:
   ```cmd
   java -version
   ```
   应该显示版本 17.x.x

---

## 第二步：安装 Flutter SDK

### 下载
1. 访问: https://docs.flutter.dev/release/archive
2. 下载最新稳定版: **flutter_windows_3.19.x-stable.zip**

### 解压
1. 创建文件夹: `C:\Development`
2. 将 zip 解压到 `C:\Development\flutter`

### 配置环境变量

**Windows 10/11:**
1. 右键"此电脑" → 属性 → 高级系统设置
2. 点击"环境变量"
3. 在"用户变量"中找到 `Path`，双击编辑
4. 点击"新建"，添加: `C:\Development\flutter\bin`
5. 点击确定保存

### 验证
打开新的命令提示符，输入:
```cmd
flutter --version
```
应该显示 Flutter 版本信息。

---

## 第三步：安装 Android Studio

### 为什么需要？
- 安卓模拟器（在电脑上运行 APP）
- Android SDK
- 开发工具

### 安装步骤

1. **下载**
   访问: https://developer.android.com/studio
   点击"Download Android Studio"

2. **安装**
   双击下载的安装程序，按向导安装

3. **首次启动配置**
   - 选择"Standard"安装
   - 接受所有许可协议
   - 等待下载组件（约 15-30 分钟）

4. **创建模拟器**
   打开 Android Studio → More Actions → Virtual Device Manager
   - 点击"Create Device"
   - 选择: Phone → Pixel 6
   - 系统镜像: 下载并选择 **Android 14.0 (API 34)**
   - 点击 Finish 创建

---

## 第四步：配置 Flutter

### 运行 Flutter Doctor

打开命令提示符，输入:
```cmd
flutter doctor
```

您会看到类似这样的输出:
```
[✓] Flutter (Channel stable, 3.19.0, ...)
[✓] Windows version
[!] Android toolchain - develop for Android devices
    ! Android SDK not found
[✓] Chrome - develop for the web
[!] Android Studio
    ✗ android-studio-dir not set
```

### 解决问题

**问题1: Android SDK 未找到**
```cmd
flutter config --android-sdk C:\Users\%USERNAME%\AppData\Local\Android\Sdk
```

**问题2: Android Studio 路径未设置**
```cmd
flutter config --android-studio-dir "C:\Program Files\Android\Android Studio"
```

**问题3: 缺少 cmdline-tools**
1. 打开 Android Studio
2. Tools → SDK Manager
3. 切换到 "SDK Tools" 标签
4. 勾选 "Android SDK Command-line Tools (latest)"
5. 点击 Apply 安装

**问题4: 缺少平台工具**
```cmd
flutter doctor --android-licenses
```
按提示接受所有许可协议。

### 再次验证
```cmd
flutter doctor
```
确保所有打勾 [✓]，没有错误 [!]

## 常见问题

### 问题1: 模拟器启动失败
**解决**:
- 确保电脑 BIOS 中启用了 Intel VT-x 或 AMD-V
- 关闭 Hyper-V（Windows 功能）
- 或使用真机调试（连接 USB，开启开发者模式）

### 问题2: 编译报错 "Connection refused"
**解决**:
- 确保后端服务已启动: http://localhost:8081
- 修改 API 地址为电脑实际 IP

### 问题3: 中文显示乱码
**解决**:
在 `main.dart` 中添加:
```dart
import 'package:flutter/material.dart';

void main() {
  WidgetsFlutterBinding.ensureInitialized();
  // 强制使用中文
  runApp(MyApp());
}
```

### 问题4: 图片加载失败
**解决**:
- 检查网络权限
- 确保后端存储服务已配置
