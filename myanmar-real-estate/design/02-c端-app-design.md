## 一、C端产品概述

### 1.1 产品定位
C端APP是缅甸房产平台面向购房者和租房者的核心入口，提供真实房源浏览、智能搜索、地图找房、在线咨询、预约带看等功能。

### 1.2 设计目标
1. **快速找房** - 让用户3步内找到目标房源
2. **建立信任** - 通过验真标识、真实评价建立安全感
3. **促成转化** - 优化IM咨询和预约带看流程
4. **本地化体验** - 完美适配缅语和缅甸用户习惯

### 1.3 页面清单

| 模块 | 页面名称 | 优先级 | 说明 |

#### 组件清单

| 组件 | 类型 | 样式说明 |
|------|------|----------|
| Logo | 图片 | 80x80px, 居中 |
| 标题 | 文字 | H1, 居中 |
| 副标题 | 文字 | Body-M, Gray-600 |
| 手机输入 | 组合输入框 | 区号+手机号，高度48px |
| 验证码输入 | 输入框+按钮 | 右侧获取验证码按钮 |
| 登录按钮 | 主按钮 | 全宽，高度48px |
| 第三方登录 | 图标按钮 | Facebook/WhatsApp图标 |
| 注册入口 | 文字链接 | Primary-700 |

#### 交互说明

1. **手机号输入**
   - 区号默认显示 +95 (缅甸)
   - 手机号自动格式化显示 (9XX XXX XXX)
   - 输入完成自动校验格式

2. **验证码获取**
   - 点击后按钮变为倒计时 (60秒)
   - 倒计时期间按钮禁用
   - 支持重新获取

3. **登录按钮**
   - 手机号和验证码都填写后才可点击
   - 点击后显示加载状态
   - 登录成功跳转首页

4. **第三方登录**
   - 点击唤起对应授权
   - 授权成功后自动登录/注册

#### 状态变化

**空态:**
- 登录按钮禁用
- 输入框显示占位符

**加载态:**
- 登录按钮显示loading图标
- 输入框禁用

**错误态:**
- 手机号错误: 输入框变红，显示错误提示"请输入正确的缅甸手机号"
- 验证码错误: 输入框变红，显示错误提示"验证码错误，还剩X次机会"

   - 引导"去发现好房源"
