## 一、Web后台产品概述

### 1.1 产品定位
Web管理后台是缅甸房产平台面向内部运营团队的核心管理工具，提供用户管理、房源审核、数据监控、系统配置等功能，支撑平台的日常运营和决策分析。

### 1.2 设计目标
1. **高效管理** - 表格化展示，支持批量操作，提升管理效率
2. **数据可视** - 数据大屏展示核心指标，辅助运营决策
3. **流程顺畅** - 审核流程清晰，状态流转明确
4. **安全可靠** - 权限分级，操作留痕，保障数据安全

### 1.3 与移动端的设计差异

| 维度 | C端/B端 APP | Web后台 |
|------|-------------|---------|
| 屏幕尺寸 | 移动端小屏 | 桌面端大屏 |
| 布局方式 | 单栏流式 | 侧边栏+内容区 |
| 信息密度 | 低密度 | 高密度 |
| 交互方式 | 触摸为主 | 鼠标键盘为主 |
| 核心组件 | 卡片、列表 | 表格、表单、图表 |

### 1.4 页面清单

| 模块 | 页面名称 | 优先级 | 说明 |
|------|----------|--------|------|
| 认证 | 登录页 | P0 | 后台登录 |
| 首页 | 数据大屏 | P0 | Dashboard看板 |
| 用户 | 用户管理 | P0 | C端用户管理 |
| | 经纪人管理 | P0 | B端经纪人管理 |
| 房源 | 房源审核 | P0 | 房源上架审核 |
| | 验真管理 | P0 | 验真任务管理 |
| | 房源列表 | P1 | 全部房源管理 |
| 数据 | 数据BI | P0 | 业务数据分析 |
| | 运营报表 | P1 | 运营数据报表 |
| 系统 | 系统设置 | P0 | 基础配置 |
| | 权限管理 | P0 | 角色权限配置 |
| | 日志管理 | P1 | 操作日志审计 |

### 3.4 交互说明

1. **账号输入**
   - 支持邮箱或用户名
   - 输入完成自动校验格式
   - 错误时边框变红

2. **密码输入**
   - 默认隐藏密码
   - 点击眼睛图标切换显示
   - 支持回车键登录

3. **登录按钮**
   - 账号密码都填写后才可点击
   - 点击后显示loading状态
   - 登录成功跳转到数据大屏

4. **错误处理**
   - 账号不存在: "账号或密码错误"
   - 密码错误: "账号或密码错误，还剩X次机会"
   - 账号禁用: "该账号已被禁用，请联系管理员"
