## 一、项目简介

### 1.1 产品概述
缅甸首个数字化房产交易与服务平台，包含：
- **C端APP**: 购房者/租房者使用 (Flutter)
- **B端APP**: 经纪人/团队长使用 (Flutter)
- **Web管理后台**: 运营/管理员使用 (Vue3)
- **后端服务**: Go + PostgreSQL + Redis

### 1.2 项目周期
- **总工期**: 8周
- **启动日期**: 2026-03-17
- **计划完成**: 2026-05-12

## 三、快速开始

### 3.1 启动项目

1. **阅读PRD**: 理解产品需求
2. **查看主计划**: 了解里程碑安排
3. **阅读任务书**: 各Agent了解职责
4. **参加启动会**: 全员对齐目标

### 3.2 每日工作

1. **查看依赖**: 确认前置任务完成
2. **执行任务**: 按计划完成开发
3. **提交日报**: 23:00前更新进度
4. **参加站会**: 同步进展与问题

### 3.3 问题升级

- 技术问题 → AGENT-001 → PM
- 进度问题 → PM
- 协作问题 → PM
