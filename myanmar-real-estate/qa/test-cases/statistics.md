## 三、测试重点与风险提示

### 3.1 核心测试重点（P0）

1. **账号安全**：注册登录、实名认证流程
2. **房源搜索**：多条件筛选、关键词搜索
3. **地图找房**：三级聚合、位置定位
4. **IM通信**：消息收发、实时性
5. **预约带看**：时间选择、双端确认
6. **验真流程**：拍照规范、产权核验
7. **ACN分佣**：角色确认、分佣计算

### 3.2 高风险功能

| 风险项 | 风险等级 | 测试建议 |
|--------|----------|----------|
| 地图聚合性能 | 高 | 大数据量压测 |
| ACN分佣计算 | 高 | 多场景验证 |
| 验真任务流 | 中 | 全流程覆盖 |
| IM实时性 | 中 | 弱网测试 |
| 支付/提现 | 高 | 安全测试 |

### 3.3 兼容性重点

- 缅甸本土机型适配（Samsung、Xiaomi等）
- 缅语Unicode显示
- 缅甸时区（UTC+6:30）处理
- 低版本Android（9+）兼容
